SmartRouterZone 聚焦智能路由与 OpenWrt。
提供固件编译、设备适配与驱动调优、源码维护、常用插件与配置集合。
覆盖 CLX S20P / WAX206 / mt798x 等设备与分支，持续更新与优化。
